Conecta con Google Workspace para sincronizar reuniones y crear eventos directamente desde el CRM.

2. Crea un OAuth 2.0 Client ID (tipo Web)
3. Añade como "Origen autorizado de JavaScript"
4. Pega el Client ID aquí y haz clic en Guardar
